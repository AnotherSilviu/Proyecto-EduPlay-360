Opción tecnológica escogida

Se destacan dos subapartados software y hardware.

  • En el apartado de software se encuentran las siguientes herramientas: plataforma central gamificada (Moodle con plugins sobre gamificación como Badges y Level Up, para gestionar la narrativa del proyecto, progresiones y recompensas, totalmente compatible con plataformas móviles fomentando así el Mobile Learning), herramientas de diseño 3D (Tinkercad y Blender para prototipado e impresión), entornos de realidad virtual (Mozilla Hubs o EngageVR para simulaciones inmersivas) y software de robótica (Blockly y Arduino IDE)
  • En los recursos hardware destacan: dispositivos de RV (por ejemplo, Oculus Quest 2 o HTC Vive Focus 3), modelos básicos de impresión 3D (como Creality Ender 3), kits de robótica básica (arduino starter kits) y los ordenadores particulares de las personas participantes.
